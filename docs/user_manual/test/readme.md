## 10. Какие правила нужно учитывать для установки комментариев к привязке?
1. Комментарий должен начинаться с заглавной буквы
2. Комментарий не должен содержать спецсимволы, скобочки, и др. (за исключением символа - №)
3. Комментарий может содержать буквы латинского и кириллического алфавита только в верхнем регистре, а так же буквенно цифровые - обозначения (прим., CR2, B4 и др.)
4. Комментарий может содержать буквы латинского и кириллического алфавита в любом регистре (верхний, нижний), а так же буквенно цифровые - обозначения (прим., CR2, B4 и др.)
5. Комментарий должен содержать в себе ОУ
